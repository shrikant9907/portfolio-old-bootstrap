# Shrikant Yadav Portfolio

Production-ready static Bootstrap portfolio for Shrikant Yadav.

## What is included

- SEO-ready one-page portfolio
- Custom inline SVG icons
- Responsive Bootstrap layout
- AOS for subtle section animation
- Isotope portfolio filtering
- GLightbox project image preview
- Person, ProfessionalService, WebSite, and FAQ schema
- robots.txt and sitemap.xml

## Deployment

Upload the folder contents to the root of `https://shrikant.shrimo.com/`.
