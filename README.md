# Shrimo Verse

Interactive portfolio universe for Shrikant Yadav / Shrimo Innovations.

## How to run
Open `index.html` directly or serve the folder with any static server.

Example:

```bash
python3 -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## Controls
Desktop:
- Guided Flight
- Free Explore
- Auto Flight
- Pause Orbit
- Return to Core
- Replay Guide
- Launch Project
- Orbit Zoom slider

Keyboard:
- G: Guide
- E: Explore
- A: Auto Flight
- Space: Pause Orbit
- R/Home: Return to Core
- +/-: Zoom
- Escape: Close panels
- Arrow Left/Right: Change universe zone

Mobile:
- Bottom control dock
- Tap particles to inspect
- Swipe left/right between zones
- Pinch zoom
