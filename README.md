# Shrikant Yadav Portfolio

Production-ready static Bootstrap portfolio for Shrikant Yadav, Web Designer and Full Stack Developer in Narmadapuram.

## Main files
- `index.html`
- `css/portfolio-style.css`
- `js/portfolio-scripts.js`
- `robots.txt`
- `sitemap.xml`
- `PORTFOLIO_UPGRADE_PLAN.md`
- `CHANGELOG.md`

## Libraries used
- Bootstrap
- jQuery / Popper for existing Bootstrap behavior
- GSAP
- AOS
- Shuffle.js
- GLightbox
- CountUp.js
- Splide
- TypeIt
