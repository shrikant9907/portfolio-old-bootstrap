# Shrimo Verse Final Feature, Impression, and Interaction Checklist

## 1. Core Impression
- [x] Experience feels like one screen / one universe, not normal website sections.
- [x] Main layout uses full viewport width and height, not a centered container layout.
- [x] Visual system uses Black + Cyan only.
- [x] First impression says “Shrimo Verse” clearly.
- [x] Loader shows “You are going to enter Shrimo Verse”.
- [x] Every visible particle has a purpose: technology, tool, product, proof, review, or contact path.
- [x] Footer is integrated as “Shrimo Base Station / Conversion Dock”, not a separate block.

## 2. Screen-First Layout
- [x] Stage fills the screen with `100dvh` / `100vw` behavior.
- [x] Header is a floating full-screen control bar, not a boxed page container.
- [x] Overlay content sits on the screen as an interface layer.
- [x] World plane scales to desktop, 1366px laptop, tablet, and mobile.
