# Shrimo Verse Final Master Prompt

Build Shrimo Verse as a production-ready, full-screen, black-and-cyan interactive portfolio universe.

The final experience must not feel like a normal website. It must feel like a real universe where `SV / Shrimo Verse` is the fixed center core and everything meaningful orbits around it.

## Must-have rules
- Pure black cinematic background.
- No grid background.
- SV core fixed in center like a sun.
- Core never overlaps headings, buttons, particles, or tooltips.
- Particle labels use only the name, not category prefixes.
- Product objects behave like orbiting gallery/showcase bodies.
- Desktop uses SVG HUD controls with tooltips.
- Mobile uses simplified bottom control dock.
- Rocket cursor works only on desktop and uses yellow/orange flame.
- Orange/yellow is only for flame, thrust, hover, and click lock-on.
- Entry screen has no Skip Intro button.
- Entry click animates into the universe.
- First Launch Guide teaches interaction.
- Auto Flight moves through Core → Technology → Product Gallery → Proof → Client Signals → Launch Dock.
- Footer is Launch Dock, not a footer section.
- Every object is meaningful.
- Real reviews only.
- No fake claims.
- Keep the build stable and production-ready.

The user should feel:
- I entered a universe.
