## Request

Move the main controls to the bottom-right like floating Instagram-style icons.

---

## Final behavior

### Layout
- Brand remains top-left.
- Main HUD controls move to bottom-right.
- Controls stack vertically.

### Visual style
- No icon backgrounds.
- No borders.
- No wrappers.
- Clean floating icon appearance.

### Interaction
- Hover/focus changes icon color to the action highlight color.
- Tooltip appears on hover/focus to the left of the icon.

### Responsive
- Desktop: bottom-right vertical floating rail.
- Tablet: compact bottom-right vertical floating rail.
- Mobile: compact bottom-right floating rail above the dock.
