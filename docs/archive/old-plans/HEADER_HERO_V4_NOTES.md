# Header + Hero V4 Notes

## Creative Direction

The first screen now uses an **AI Product Studio** concept instead of a dashboard-heavy command console. The goal is to make the hero feel more memorable, cinematic, and premium while keeping the business conversion path clear.

## Header

- Dark transparent glass navigation.
- Compact SY Studio mark.
- Shorter navigation: Studio, Work, Services, Proof, Reviews, Contact.
- Strong but compact Start Project CTA.
- Scanner-style active navigation interaction.

## Hero

- New headline: “I design and build digital systems that feel ahead.”
- Shorter, more creative positioning copy.
- TypeIt line focuses on what is being shaped now.
- CTA buttons: Enter the Work and Start a Project.
- Proof line stays visible but clean.

## Visual System

- Custom inline SVG crystal core.
- Idea → Design → Build → Launch motion path.
- Floating capability nodes: Websites, Next.js, WordPress, SEO, UI/UX, Web Apps.
- Deep navy background with coral and cyan energy accents.
- Less dashboard, less small text, more one-visual-anchor creativity.
